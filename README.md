# E-learn - Website & Android Aplication

E-Learning System with Face Recognition & Eye Tracking is an integrated digital learning system that combines Artificial Intelligence technology to improve learning effectiveness and student attendance. This system is built using FastAPI (Python) as the backend connected to a MySQL database, React with Vite and TailwindCSS for the web frontend, and React Native for the Android mobile application.


Features of this Android app:
* Face Detection For Presention
* Eye tracking For Material Assessment
* Profile
* Information
* Notification Real Time
